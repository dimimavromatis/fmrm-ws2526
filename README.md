# fmrm-ws-2526
This repository contains code developed for the lecture Financial Markets and Risk Management (WS 2025/26) at the Department of Finance, University of Stuttgart. It comprises Manim-based visualizations and Python implementations of key quantitative finance models, including option pricing via binomial trees, CRR parametrization, convergence analysis, and comparisons with the Black–Scholes framework.

Furthermore, the repository includes a case study on EUR/USD exchange rate risk for BMW, employing advanced mathematical techniques such as geometric Brownian motion, Monte Carlo simulation, and constrained optimization to model and analyze corporate financial risk ([see notebook](/LBP_04.ipynb)).
